# Espacio Psicopedagógico - Landing

Sitio estático para publicar con GitHub Pages.

## Cómo subirlo

1. Crear un repositorio en GitHub.
2. Subir estos archivos en la raíz del repositorio:
   - index.html
   - styles.css
   - README.md
3. Ir a Settings > Pages.
4. En Source, elegir Deploy from a branch.
5. Elegir la rama main y carpeta /root.
6. Guardar.

La página quedará publicada en una dirección similar a:
https://TU-USUARIO.github.io/NOMBRE-DEL-REPOSITORIO/
